# Number Guessing Game

A console-based number guessing game built with Dart.

## Features

- 3 difficulty levels (Easy, Medium, Hard)
- Smart hint system
- Score tracking
- Statistics
- Leaderboard

## How to Run

```bash
dart Mini_Game.dart
```

## How to Play

1. Select difficulty
2. Guess the secret number
3. Follow the hints
4. Beat your high score!
